# pomodoro 
**A personal project**

An Arduino implementation of a [pomodoro](https://en.wikipedia.org/wiki/Pomodoro_Technique "pomodoro") timer, written for the _Arduino Uno Rev3_.\
Demo: https://www.youtube.com/watch?v=ZWuYBsaMIgg
